# 18xx Helper

Dividend & final-score calculator for 18xx-style train games (1817 / 1862 / 18Korea).

**Use it here → https://tabletop-chronicler.github.io/18xx-helper/**

- Dividend payouts per player, short positions included
- Per-game stock price movement hints
- Final scoring: cash + shares × current price, with ranking
- Live session sharing (one person banks, everyone watches)
- Korean / English

Single self-contained HTML file — no build, no dependencies.
